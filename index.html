The difference in session storage vs local storage?

The main difference between Local storage and Session storage is local storage data is permanently shown in our browser until. users will be deleting this data. Local storage store is 10 megabytes and session storage is 5 megabytes. Local storage and session storage has same syntax. But the difference is the localStorage variable and sessionStorage variable. Local storage is permanent session But session storage is tied to the particular session.

What does javascript Event loop?

Event loop হল জাভাস্ক্রিপ্টের asynchronous সিস্টেমে কাজ করার একটি মাধ্যম । জাভাস্ক্রিপ্টে এ asynhronous অপারেশন চালানোর জন্য Web API ব্যাপক ভূমিকা পালন করে । আমরা যখন কোনো function কে call করি সেটা call stack এ যায় এবং সেখান থেকে রান হয় । এরপর যদি কোনো callstack এ asynchronous behaviour দেখে | তখন সেটা web API এ পাঠিয়ে দেয় । এরপর web API থেকে ডাটা callback queue তে পাঠিয়ে দেয় । মূলত এখানেই event loop কাজ করে যখন দেখে call stack এ আর কোনো কাজ নেই । তখন callback queue থেকে ডাটা call stack এ পাঠিয়ে দেয় । মূলত, এভাবেই Event loop asynchronous সিস্টেম এ কাজ করে|
